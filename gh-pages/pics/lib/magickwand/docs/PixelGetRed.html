PixelGetRed

PixelGetRed -- returns the normalized red color value of the PixelWand

Description

float PixelGetRed( PixelWand pxl_wnd ) 

Returns the normalized red color value of the PixelWand.

The returned value is in the range 0 <= red <= 1.

A description of each parameter follows:

pxl_wnd A PixelWand resource
