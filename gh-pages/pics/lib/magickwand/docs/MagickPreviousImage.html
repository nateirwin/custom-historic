MagickPreviousImage

MagickPreviousImage -- causes the previous image (if any) in the MagickWand's image list to become the active/current image

Description

bool MagickPreviousImage( MagickWand mgck_wnd ) 

Causes the previous image (if any) in the MagickWand's image list to
become the active/current image.

If successful, returns TRUE; if there are no more images preceeding the
current one in the MagickWand's image list, returns FALSE.

A description of each parameter follows:

mgck_wnd A MagickWand resource
