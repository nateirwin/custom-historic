DrawSetTextDecoration

DrawSetTextDecoration -- sets the decoration applied when annotating with text

Description

void DrawSetTextDecoration( DrawingWand drw_wnd, int decoration_type ) 

Sets the decoration applied when annotating with text.

Returns void.

decoration_type must be a DecorationType constant, one of the following:
   MW_NoDecoration
   MW_UnderlineDecoration
   MW_OverlineDecoration
   MW_LineThroughDecoration

A description of each parameter follows:

drw_wnd A DrawingWand resource
decoration_type A DecorationType constant
