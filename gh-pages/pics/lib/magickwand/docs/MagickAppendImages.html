MagickAppendImages

MagickAppendImages -- appends a set of images

Description

MagickWand MagickAppendImages( MagickWand mgck_wnd [, bool stack_vertical] ) 

Appends a set of images.

If successful, the returned MagickWand will contain a single-frame image,
the result of the operation.

FALSE is returned otherwise.

A description of each parameter follows:

mgck_wnd A MagickWand resource
stack_vertical (OPTIONAL) By default, images are stacked left-to-right.
Set stack_vertical to TRUE to stack them top-to-bottom
