PixelSetYellowQuantum

PixelSetYellowQuantum -- sets the yellow Quantum color of the PixelWand

Description

void PixelSetYellowQuantum( PixelWand pxl_wnd, float yellow ) 

Sets the yellow Quantum color value of the PixelWand.

The yellow Quantum color value must be in the range 0 <= yellow <=
MW_MaxRGB.

A description of each parameter follows:

pxl_wnd A PixelWand resource
yellow The yellow Quantum color value
