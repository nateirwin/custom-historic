MagickQueryConfigureOptions

MagickQueryConfigureOptions -- returns any configure options that match the specified pattern

Description

array MagickQueryConfigureOptions( string pattern ) 

Returns any configure options that match the specified pattern (e.g. '*'
for all).

Options include NAME, VERSION, LIB_VERSION, etc.

A description of each parameter follows:

pattern A text string containing a pattern
