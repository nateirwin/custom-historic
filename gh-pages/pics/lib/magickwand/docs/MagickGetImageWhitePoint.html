MagickGetImageWhitePoint

MagickGetImageWhitePoint -- returns the chromaticy white point

Description

array MagickGetImageWhitePoint( MagickWand mgck_wnd ) 

returns the chromaticy white point.

The returned array is defined as follows:
Array
{
   [0] => the chromaticity white x-point.
   [1] => the chromaticity white y-point.
}

A description of each parameter follows:

mgck_wnd A MagickWand resource
