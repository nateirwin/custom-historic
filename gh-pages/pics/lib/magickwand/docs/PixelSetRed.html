PixelSetRed

PixelSetRed -- sets the normalized red color value of the PixelWand

Description

void PixelSetRed( PixelWand pxl_wnd, float red ) 

Sets the normalized red color value of the PixelWand.

The red color value must be in the range 0 <= red <= 1.

A description of each parameter follows:

pxl_wnd A PixelWand resource
red The normalized red color value
