MagickPaintTransparentImage

MagickPaintTransparentImage -- changes the opacity of any pixel in the current active image that matches the target color

Description

bool MagickPaintTransparentImage( MagickWand mgck_wnd, PixelWand target [, float opacity [, float fuzz]] ) 

Changes the opacity of any pixel in the current active image that matches
the target color.

A description of each parameter follows:

mgck_wnd A MagickWand resource
target Change any pixel matching this target PixelWand's color to the specified opacity value within the image.
opacity (OPTIONAL) the replacement opacity Quantum value (it must conform to 0 <= opacity <= MW_MaxRGB); it is set to MW_TransparentOpacity by default.
fuzz (OPTIONAL) 0 by default.
By default target must match a particular pixel color exactly.
However, in many cases two colors may differ by a small amount.
The fuzz member of image defines how much tolerance is acceptable to consider two colors as the same.
For example, set fuzz to 10 and the color red at intensities of 100 and 102 respectively are now interpreted as the same color for the purposes of the floodfill
