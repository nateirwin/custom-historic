PixelSetAlphaQuantum

PixelSetAlphaQuantum -- sets the alpha Quantum color of the PixelWand

Description

void PixelSetAlphaQuantum( PixelWand pxl_wnd, float alpha ) 

Sets the alpha Quantum color value of the PixelWand.

The alpha Quantum color value must be in the range 0 <= alpha <=
MW_MaxRGB.

A description of each parameter follows:

pxl_wnd A PixelWand resource
alpha The alpha Quantum color value
