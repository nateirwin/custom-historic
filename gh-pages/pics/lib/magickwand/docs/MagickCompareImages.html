MagickCompareImages

MagickCompareImages -- compares one or more images, or image channels and returns the specified distortion metric

Description

array MagickCompareImages( MagickWand mgck_wnd, MagickWand reference_wnd, int metric_type [, int channel_type] ) 

Compares one or more images, or image channels and returns the specified
distortion metric.

Returns an array, or FALSE if an error occurred.

The returned array is defined as follows:
Array
{
   [0] => a MagickWand containing the image result of the comparison
          operation

   [1] => the computed distortion metric
}

metric_type must be be a MetricType constant, one of the following:
   MW_UndefinedMetric                  MW_MeanAbsoluteErrorMetric
   MW_MeanSquaredErrorMetric           MW_PeakAbsoluteErrorMetric
   MW_PeakSignalToNoiseRatioMetric     MW_RootMeanSquaredErrorMetric

If the optional channel_type parameter is specified, calls
MagickCompareImageChannels() on the specified image channels.

channel_type must be be a ChannelType constant, one of the following:
   MW_RedChannel        MW_CyanChannel
   MW_GreenChannel      MW_MagentaChannel
   MW_BlueChannel       MW_YellowChannel
   MW_AlphaChannel      MW_OpacityChannel
   MW_BlackChannel      MW_IndexChannel
   MW_AllChannels

A description of each parameter follows:

mgck_wnd A MagickWand resource
reference_wnd The reference MagickWand resource
metric_type A MetricType constant
channel_type (OPTIONAL) ChannelType constant
