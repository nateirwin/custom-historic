PixelGetAlpha

PixelGetAlpha -- returns the normalized alpha color value of the PixelWand

Description

float PixelGetAlpha( PixelWand pxl_wnd ) 

Returns the normalized alpha color value of the PixelWand.

The returned value is in the range 0 <= alpha <= 1.

A description of each parameter follows:

pxl_wnd A PixelWand resource
