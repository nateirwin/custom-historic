MagickSetImageBias -- sets the image bias for any method that convolves the current active image

Description

bool MagickSetImageBias( MagickWand mgck_wnd, float bias ) 

Sets the image bias for any method that convolves the current active image
(e.g. MagickConvolveImage()).

A description of each parameter follows:

mgck_wnd A MagickWand resource
bias The image bias
