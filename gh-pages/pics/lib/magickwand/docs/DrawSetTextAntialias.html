DrawSetTextAntialias

DrawSetTextAntialias -- controls whether text is antialiased

Description

void DrawSetTextAntialias( DrawingWand drw_wnd [, bool text_antialias] ) 

Controls whether text is antialiased.

Text is antialiased by default.

Also if "text_antialias" argument is not specified, antialiasing is turned
on.

A description of each parameter follows:

drw_wnd A DrawingWand resource
text_antialias (OPTIONAL) set to FALSE to disable antialiasing.
If not set or TRUE, will turn on antialiasing
