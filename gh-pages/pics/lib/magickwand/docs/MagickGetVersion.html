MagickGetVersion

MagickGetVersion -- returns the ImageMagick API version as a string constant and as a number

Description

array MagickGetVersion( void ) 

Returns the ImageMagick API version as a string constant and as a number.

The returned array is defined as follows:
Array
{
   [0] => version as a string
   [1] => version as a number
}
