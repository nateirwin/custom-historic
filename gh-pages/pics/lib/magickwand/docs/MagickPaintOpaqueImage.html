MagickPaintOpaqueImage

MagickPaintOpaqueImage -- changes the color of any pixel in the current active image that matches a target color

Description

bool MagickPaintOpaqueImage( MagickWand mgck_wnd, PixelWand target_pxl_wnd, PixelWand fill_pxl_wnd [, float fuzz] ) 

Changes the color of any pixel in the current active image that matches
the target color, with the color defined by fill.

A description of each parameter follows:

mgck_wnd A MagickWand resource
target_pxl_wnd Change any pixel matching this target PixelWand's color to the fill PixelWand's color within the image.
fill_pxl_wnd The fill color PixelWand.
fuzz (OPTIONAL) 0 by default.
By default target must match a particular pixel color exactly.
However, in many cases two colors may differ by a small amount.
The fuzz argument defines how much tolerance is acceptable to consider two colors as the same.
For example, set fuzz to 10 and the color red at intensities of 100 and 102 respectively are now interpreted as the same color for the purposes of the floodfill
