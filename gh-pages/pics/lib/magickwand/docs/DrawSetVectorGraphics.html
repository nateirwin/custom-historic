DrawSetVectorGraphics

DrawSetVectorGraphics -- sets the vector graphics string associated with the specified DrawingWand

Description

bool DrawSetVectorGraphics( DrawingWand drw_wnd, string vector_graphics ) 

Sets the vector graphics string associated with the specified DrawingWand.

In order to clear the DrawingWand's current commands, use:

   DrawSetVectorGraphics( $drw_wnd, "" );

Use this method with DrawGetVectorGraphics() as a method to save and load
the vector graphics state.

A description of each parameter follows:

drw_wnd A DrawingWand resource
vector_graphics The vector graphics string
