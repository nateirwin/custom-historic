DrawAffine

DrawAffine -- adjusts the current affine transformation matrix

Description

void DrawAffine( DrawingWand drw_wnd, float sx, float sy, float rx, float ry, float tx, float ty ) 

Adjusts the current affine transformation matrix with the specified affine
transformation matrix.

Note that the current affine transform is adjusted rather than replaced.

A description of each parameter follows:

drw_wnd A DrawingWand resource
sx Affine matrix parameter
sy Affine matrix parameter
rx Affine matrix parameter
ry Affine matrix parameter
tx Affine matrix parameter
ty Affine matrix parameter
