MagickMosaicImages

MagickMosaicImages -- inlays the MagickWand's image sequence to form a single coherent picture

Description

MagickWand MagickMosaicImages( MagickWand mgck_wnd ) 

Inlays the MagickWand's image sequence to form a single coherent picture.

It returns a MagickWand with each image in the sequence composited at the
location defined by the page offset of the image.

A description of each parameter follows:

mgck_wnd A MagickWand resource
