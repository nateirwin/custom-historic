MagickGetImageBlob

MagickGetImageBlob -- returns the current active image as a BLOB

Description

string MagickGetImageBlob( MagickWand mgck_wnd ) 

Returns the BLOB (Binary Large OBject) of the current active image in the
MagickWand (as a string), or explicit FALSE (suitable for === testing), if
an error occurred.

The format of the image determines the format of the returned blob (GIF,
JPEG, PNG, etc.).

To return the image BLOB in a different image format, use
MagickSetImageFormat().

It is recommended that you do NOT use this function, as the returned BLOB
is actually duplicated to PHP's internal memory.

So, at least for a short period of time, there will be 2 identical BLOBs
consuming PHP's memory.

If you want to output the image BLOB, use MagickEchoImageBlob(), and if
you want to dump the image to a file, use MagickWriteImage() or
MagickWriteImageFile().

A description of each parameter follows:

mgck_wnd A MagickWand resource
