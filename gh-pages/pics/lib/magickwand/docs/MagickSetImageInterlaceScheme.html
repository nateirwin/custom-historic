MagickSetImageInterlaceScheme

MagickSetImageInterlaceScheme -- sets the image interlace scheme of the current active image

Description

bool MagickSetImageInterlaceScheme( MagickWand mgck_wnd, int interlace_type ) 

Sets the image interlace scheme of the current active image.

Returns TRUE on success, FALSE, if an error occurred.

interlace_type must be an InterlaceType constant, one of the following:
   MW_NoInterlace
   MW_LineInterlace
   MW_PlaneInterlace
   MW_PartitionInterlace

A description of each parameter follows:

mgck_wnd A MagickWand resource
interlace_type An InterlaceType constant
