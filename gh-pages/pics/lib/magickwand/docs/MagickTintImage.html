MagickTintImage

MagickTintImage -- applies a color vector to each pixel in the current active image

Description

bool MagickTintImage( MagickWand mgck_wnd, PixelWand tint_pxl_wnd, PixelWand opacity_pxl_wnd ) 

Applies a color vector to each pixel in the current active image.

The length of the vector is 0 for black and white and at its maximum for
the midtones.

The vector weighting function is f(x)=(1-(4.0*((x-0.5)*( x-0.5)))).

A description of each parameter follows:

mgck_wnd A MagickWand resource
tint_pxl_wnd The tint color PixelWand
opacity_pxl_wnd The opacity PixelWand
