PixelSetBlack

PixelSetBlack -- sets the normalized black color value of the PixelWand

Description

void PixelSetBlack( PixelWand pxl_wnd, float black ) 

Sets the normalized black color value of the PixelWand.

The black color value must be in the range 0 <= black <= 1.

A description of each parameter follows:

pxl_wnd A PixelWand resource
black The normalized black color value
