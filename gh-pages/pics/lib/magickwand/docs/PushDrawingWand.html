PushDrawingWand

PushDrawingWand -- clones the current DrawingWand to create a new DrawingWand

Description

void PushDrawingWand( DrawingWand drw_wnd ) 

Clones the current DrawingWand to create a new DrawingWand, which is then
added to the DrawingWand stack.

The original drawing DrawingWand(s) may be returned to by invoking
PopDrawingWand().

The DrawingWands are stored on a DrawingWand stack.

For every Pop there must have already been an equivalent Push.

A description of each parameter follows:

drw_wnd A DrawingWand resource
