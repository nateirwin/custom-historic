MagickSolarizeImage

MagickSolarizeImage -- applies a special effect to the current active image

Description

bool MagickSolarizeImage( MagickWand mgck_wnd, float threshold ) 

Applies a special effect to the current active image, similar to the
effect achieved in a photo darkroom by selectively exposing areas of photo
sensitive paper to light.

Threshold range is 0 <= threshold <= MW_MaxRGB and is a measure of
the extent of the solarization.

A description of each parameter follows:

mgck_wnd A MagickWand resource
threshold Define the extent of the solarization
