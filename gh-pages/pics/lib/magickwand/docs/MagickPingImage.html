MagickPingImage

MagickPingImage -- efficiently retrieves an image file's attributes

Description

bool MagickPingImage( MagickWand mgck_wnd, string filename ) 

MagickPingImage() is like MagickReadImage() except the only valid
information returned is the image width, height, size, and format.

It is designed to efficiently obtain this information from a file without
reading the entire image sequence into memory.

A description of each parameter follows:

mgck_wnd A MagickWand resource
filename The image filename
