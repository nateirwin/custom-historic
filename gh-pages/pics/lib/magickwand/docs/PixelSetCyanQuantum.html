PixelSetCyanQuantum

PixelSetCyanQuantum -- sets the cyan Quantum color of the PixelWand

Description

void PixelSetCyanQuantum( PixelWand pxl_wnd, float cyan ) 

Sets the cyan Quantum color value of the PixelWand.

The cyan Quantum color value must be in the range 0 <= cyan <=
MW_MaxRGB.

A description of each parameter follows:

pxl_wnd A PixelWand resource
cyan The cyan Quantum color value
