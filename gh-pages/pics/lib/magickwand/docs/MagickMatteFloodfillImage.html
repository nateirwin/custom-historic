MagickMatteFloodfillImage

MagickMatteFloodfillImage -- changes the transparency value of any pixel that matches target and is an immediate neighbor

Description

bool MagickMatteFloodfillImage( MagickWand mgck_wnd, Quantum opacity, float fuzz, PixelWand bordercolor_pxl_wnd, int x, int y ) 

Changes the transparency value of any pixel that matches target and is an
immediate neighbor.

If the method FillToBorderMethod is specified, the transparency value is
changed for any neighbor pixel that does not match the bordercolor member
of image.

A description of each parameter follows:

mgck_wnd A MagickWand resource
opacity The opacity Quantum value (it must be in the range 0 <= opacity <= MW_MaxRGB).
fuzz By default target must match a particular pixel color exactly.
However, in many cases two colors may differ by a small amount.
The fuzz member of image defines how much tolerance is acceptable to consider two colors as the same.
For example, set fuzz to 10 and the color red at intensities of 100 and 102 respectively are now interpreted as the same color for the purposes of the floodfill.
bordercolor_pxl_wnd The border color PixelWand.
x, y The starting location of the operation
