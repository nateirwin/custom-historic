MagickModulateImage

MagickModulateImage -- adjust the brightness, saturation, and hue of the current active image

Description

bool MagickModulateImage( MagickWand mgck_wnd, float brightness, float saturation, float hue ) 

Adjust the brightness, saturation, and hue of the current active image.

A description of each parameter follows:

mgck_wnd A MagickWand resource
brightness The percent change in brighness (-100 thru +100).
saturation The percent change in saturation (-100 thru +100 )
hue The percent change in hue (-100 thru +100 )
