MagickGetImageResolution

MagickGetImageResolution -- returns the X and Y resolution of the current active image

Description

array MagickGetImageResolution( MagickWand mgck_wnd ) 

Returns the X and Y resolution of the current active image.

The returned array is defined as follows:
Array
{
   [0] => the image's x-resolution.
   [1] => the image's y-resolution.
}

A description of each parameter follows:

mgck_wnd A MagickWand resource
