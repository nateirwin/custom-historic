MagickSetImage -- replaces the active/current image

Description

bool MagickSetImage( MagickWand mgck_wnd, MagickWand replace_wand ) 

Replaces the active/current image with the images from the specified
MagickWand.

A description of each parameter follows:

mgck_wnd A MagickWand resource
replace_wand The MagickWand containing the replacement images
