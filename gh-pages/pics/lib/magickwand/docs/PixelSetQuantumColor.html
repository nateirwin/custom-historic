PixelSetQuantumColor

PixelSetQuantumColor -- sets the red, green, blue, and optional opacity Quantum color values of the PixelWand

Description

void PixelSetQuantumColor( PixelWand pxl_wnd, float red, float green, float blue [, float opacity] ) 

Sets the red, green, blue, and optional opacity Quantum color values of
the PixelWand.

The Quantum color values must be in the range 0 <= color <=
MW_MaxRGB.

A description of each parameter follows:

pxl_wnd A PixelWand resource
red The red Quantum color value
green The green Quantum color value
blue The blue Quantum color value
opacity The opacity Quantum color value
