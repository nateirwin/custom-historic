PixelResetIterator -- resets the PixelIterator

Description

void PixelResetIterator( PixelIterator pxl_iter ) 

Resets the PixelIterator.

Use it in conjunction with PixelGetNextIteratorRow() to iterate over all
the PixelWands in a PixelIterator.

A description of each parameter follows:

pxl_iter A PixelIterator resource
