MagickGetImageDispose

MagickGetImageDispose -- returns the image disposal method of the current active image

Description

int MagickGetImageDispose( MagickWand mgck_wnd ) 

Returns the image disposal method of the current active image.

The returned value will be a DisposeType constant, one of the following:
   MW_UnrecognizedDispose
   MW_UndefinedDispose
   MW_NoneDispose
   MW_BackgroundDispose
   MW_PreviousDispose

A description of each parameter follows:

mgck_wnd A MagickWand resource
