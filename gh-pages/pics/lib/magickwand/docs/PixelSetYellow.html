PixelSetYellow

PixelSetYellow -- sets the normalized yellow color value of the PixelWand

Description

void PixelSetYellow( PixelWand pxl_wnd, float yellow ) 

Sets the normalized yellow color value of the PixelWand.

The yellow color value must be in the range 0 <= yellow <= 1.

A description of each parameter follows:

pxl_wnd A PixelWand resource
yellow The normalized yellow color value
