MagickSetWandSize

MagickSetWandSize -- sets the size of a MagickWand

Description

bool MagickSetWandSize( MagickWand mgck_wnd, int columns, int rows ) 

Sets the size of a MagickWand.

Set it before you read a raw image format such as RGB, GRAY, or CMYK.

A description of each parameter follows:

mgck_wnd A MagickWand resource
columns The desired width in pixels.
rows The desired height in pixels
