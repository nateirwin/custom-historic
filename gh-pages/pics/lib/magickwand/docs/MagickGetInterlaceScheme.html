MagickGetInterlaceScheme

MagickGetInterlaceScheme -- returns the image interlace scheme of a MagickWand

Description

int MagickGetInterlaceScheme( MagickWand mgck_wnd ) 

Returns the image interlace scheme of a MagickWand.

The returned value will be an InterlaceType constant, one of the following:
   MW_UndefinedInterlace
   MW_NoInterlace
   MW_LineInterlace
   MW_PlaneInterlace
   MW_PartitionInterlace

A description of each parameter follows:

mgck_wnd A MagickWand resource
