MagickGetExceptionString

MagickGetExceptionString -- returns the description of a MagickWand error

Description

string MagickGetExceptionString( MagickWand mgck_wnd ) 

Returns a string description of errors / exceptions that occur when using
other MagickWand methods in the API.

If there is no error present, an empty string ('') is returned.

A description of each parameter follows:

mgck_wnd A MagickWand resource
