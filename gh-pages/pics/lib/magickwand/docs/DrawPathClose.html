DrawPathClose

DrawPathClose -- closes the current path

Description

void DrawPathClose( DrawingWand drw_wnd ) 

Adds a path element to the current path which closes the current subpath
by drawing a straight line from the current point to the current subpath's
most recent starting point (usually, the most recent move-to point).

A description of each parameter follows:

drw_wnd A DrawingWand resource
