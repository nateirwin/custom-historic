MagickSetImageIterations

MagickSetImageIterations -- sets the number of iterations of the current active image

Description

bool MagickSetImageIterations( MagickWand mgck_wnd, float iterations ) 

Sets the number of iterations of the current active image.

A description of each parameter follows:

mgck_wnd A MagickWand resource
iterations The desired number of iterations (loops) the image should cycle through
