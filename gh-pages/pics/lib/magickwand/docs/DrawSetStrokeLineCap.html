DrawSetStrokeLineCap

DrawSetStrokeLineCap -- sets the shape to be used at the end of open subpaths when they are stroked

Description

void DrawSetStrokeLineCap( DrawingWand drw_wnd, int line_cap ) 

Sets the shape to be used at the end of open subpaths when they are
stroked.

Returns void.

line_cap must be a LineCap constant, one of the following:
   MW_ButtCap
   MW_RoundCap
   MW_SquareCap

A description of each parameter follows:

drw_wnd A DrawingWand resource
line_cap A LineCap constant
