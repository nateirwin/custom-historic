PixelSetIteratorRow

PixelSetIteratorRow -- set the current row of the PixelIterator

Description

bool PixelSetIteratorRow( PixelIterator pxl_iter, int row ) 

Sets the current row of the PixelIterator.

Valid row values are based on the image on which the PixelIterator was
based.

Valid row values are in the range y <= row < height

where
   y == the y ordinate of the origin of the region on which the
        PixelIterator was based, relative to the image from where the
        region originated.
        In terms of PixelIterators created with NewPixelIterator(),
        y == 0; for those created with NewPixelRegionIterator(),
        y == the y argument.

   height == the height of the region on which the PixelIterator was
             based.
             In terms of PixelIterators created with NewPixelIterator(),
             height == image height; for those created with
             NewPixelRegionIterator(), height == the rows argument.

Will set the row and retun TRUE if successful, FALSE if otherwise (if the
row value is invalid).

Please note: This function DOES NOT set any exceptions.

A description of each parameter follows:

pxl_iter A PixelIterator resource
row The desired PixelIterator row
