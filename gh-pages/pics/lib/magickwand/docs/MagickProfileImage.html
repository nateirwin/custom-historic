MagickProfileImage

MagickProfileImage -- adds or removes a ICC, IPTC, or generic profile from the current active image

Description

bool MagickProfileImage( MagickWand mgck_wnd, string name [, string profile] ) 

Adds or removes a ICC, IPTC, or generic profile from the current active
image.

If the profile is NULL, it is removed from the current active image;
otherwise it is added.

Use a name of '*' and a profile of NULL to remove all profiles from the
current active image.

A description of each parameter follows:

mgck_wnd A MagickWand resource
name Name of profile to add or remove: ICC, IPTC, or generic profile.
profile (OPTIONAL) the profile; is NULL by default (for removing a profile/profiles)
