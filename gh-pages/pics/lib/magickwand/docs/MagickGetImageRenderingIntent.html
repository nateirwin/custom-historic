MagickGetImageRenderingIntent

MagickGetImageRenderingIntent -- returns the image rendering intent of the current active image

Description

int MagickGetImageRenderingIntent( MagickWand mgck_wnd ) 

Returns the image rendering intent of the current active image.

The returned value will be a RenderingIntent constant, one of the
following:
   MW_UndefinedIntent
   MW_SaturationIntent
   MW_PerceptualIntent
   MW_AbsoluteIntent
   MW_RelativeIntent

A description of each parameter follows:

mgck_wnd A MagickWand resource
