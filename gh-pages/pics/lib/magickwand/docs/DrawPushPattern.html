DrawPushPattern

DrawPushPattern -- starts the definition of a named pattern

Description

bool DrawPushPattern( DrawingWand drw_wnd, string pattern_id, float x, float y, float width, float height ) 

DrawPushPattern() indicates that subsequent commands up to a
DrawPopPattern() command comprise the definition of a named pattern.

The pattern space is assigned top left corner coordinates, a width and
height, and becomes its own drawing space.

Anything which can be drawn may be used in a pattern definition.

Named patterns may be used as stroke or brush definitions.

A description of each parameter follows:

drw_wnd A DrawingWand resource
pattern_id Pattern identification for later reference
x X ordinate of top left corner
y Y ordinate of top left corner
width Width of pattern space
height Height of pattern space
