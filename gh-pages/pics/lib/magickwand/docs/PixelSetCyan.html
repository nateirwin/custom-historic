PixelSetCyan

PixelSetCyan -- sets the normalized cyan color value of the PixelWand

Description

void PixelSetCyan( PixelWand pxl_wnd, float cyan ) 

Sets the normalized cyan color value of the PixelWand.

The cyan color value must be in the range 0 <= cyan <= 1.

A description of each parameter follows:

pxl_wnd A PixelWand resource
cyan The normalized cyan color value
