MagickSwirlImage

MagickSwirlImage -- swirls the pixels about the center of the current active image

Description

bool MagickSwirlImage( MagickWand mgck_wnd, float degrees ) 

Swirls the pixels about the center of the current active image, where
degrees indicates the sweep of the arc through which each pixel is moved.

You get a more dramatic effect as the degrees move from 1 to 360.

A description of each parameter follows:

mgck_wnd A MagickWand resource
degrees Define the tightness of the swirling effect
