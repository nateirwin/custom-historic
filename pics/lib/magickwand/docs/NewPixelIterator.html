NewPixelIterator

NewPixelIterator -- returns a new PixelIterator resource.

Description

PixelIterator NewPixelIterator( MagickWand mgck_wnd ) 

Returns a new PixelIterator resource, that is based on the entire current
active image of the specified MagickWand.

The number of rows and columns in the PixelIterator get set to the height
and width (respectively) of the current active image of the MagickWand
argument (mgck_wnd).

A description of each parameter follows:

mgck_wnd The MagickWand that the Pixeliterator is to be derived from
