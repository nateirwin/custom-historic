MagickSetImageColormapColor

MagickSetImageColormapColor -- sets the color of the specified colormap index of the current active image

Description

bool MagickSetImageColormapColor( MagickWand mgck_wnd, float index, PixelWand mapcolor_pxl_wnd ) 

Sets the color of the specified colormap index of the current active image.

A description of each parameter follows:

mgck_wnd A MagickWand resource
index The offset into the image colormap.
mapcolor_pxl_wnd The desired new color will be retrieved from this PixelWand
