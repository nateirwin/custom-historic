MagickMontageImage

MagickMontageImage -- creates a composite image by combining several separate images

Description

MagickWand MagickMontageImage( MagickWand mgck_wnd, DrawingWand drw_wnd, string tile_geometry, string thumbnail_geometry, int montage_mode, string frame ) 

Creates a composite image by combining several separate images.

The images are tiled on the composite image with the name of the image
optionally appearing just below the individual tile.

Returns a MagickWand resource containing the composite image on success,
or FALSE if an error occurs.

montage_mode must be a MontageMode constant, one of the following:
   MW_FrameMode
   MW_UnframeMode
   MW_ConcatenateMode

A description of each parameter follows:

mgck_wnd The MagickWand resource.
drw_wnd A DrawingWand resource.
The font name, size, and color are obtained from this.
tile_geometry An ImageMagick geometry string, specifying the number of tiles per row and page (e.g. "6x4+0+0").
thumbnail_geometry An ImageMagick geometry string, specifying the preferred image size and border size of each thumbnail (e.g. "120x120+4+3").
montage_mode The thumbnail's framing mode -- a MontageMode constant (MW_FrameMode, MW_UnframeMode, or MW_ConcatenateMode)
frame An ImageMagick geometry string, specifying an ornamental border (e.g. "15x15+3+3") that will surround the image.
The frame color is that of the thumbnail's matte color
