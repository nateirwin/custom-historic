PixelGetGreenQuantum

PixelGetGreenQuantum -- returns the green Quantum color of the PixelWand

Description

float PixelGetGreenQuantum( PixelWand pxl_wnd ) 

Returns the green Quantum color value of the PixelWand.

The returned value is in the range 0 <= green <= MW_MaxRGB.

A description of each parameter follows:

pxl_wnd A PixelWand resource
