MagickSetInterlaceScheme

MagickSetInterlaceScheme -- sets the interlace scheme for the MagickWand's image sequence

Description

bool MagickSetInterlaceScheme( MagickWand mgck_wnd, InterlaceType interlace_type ) 

Sets the interlace scheme for the MagickWand's image sequence.

Returns TRUE on success, FALSE, if an error occurred.

interlace_type must be an InterlaceType constant, one of the following:
   MW_NoInterlace
   MW_LineInterlace
   MW_PlaneInterlace
   MW_PartitionInterlace

A description of each parameter follows:

mgck_wnd A MagickWand resource
interlace_type An InterlaceType constant
