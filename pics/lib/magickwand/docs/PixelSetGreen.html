PixelSetGreen

PixelSetGreen -- sets the normalized green color value of the PixelWand

Description

void PixelSetGreen( PixelWand pxl_wnd, float green ) 

Sets the normalized green color value of the PixelWand.

The green color value must be in the range 0 <= green <= 1.

A description of each parameter follows:

pxl_wnd A PixelWand resource
green The normalized green color value
