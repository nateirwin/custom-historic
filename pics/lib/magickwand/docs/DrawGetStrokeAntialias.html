DrawGetStrokeAntialias

DrawGetStrokeAntialias -- returns the current stroke antialias setting

Description

bool DrawGetStrokeAntialias( DrawingWand drw_wnd ) 

Returns the current stroke antialias setting.

Stroked outlines are antialiased by default.

When antialiasing is disabled stroked pixels are thresholded to determine
if the stroke color or underlying canvas color should be used.

A description of each parameter follows:

drw_wnd A DrawingWand resource
