PixelSetBlueQuantum

PixelSetBlueQuantum -- sets the blue Quantum color of the PixelWand

Description

void PixelSetBlueQuantum( PixelWand pxl_wnd, float blue ) 

Sets the blue Quantum color value of the PixelWand.

The blue Quantum color value must be in the range 0 <= blue <=
MW_MaxRGB.

A description of each parameter follows:

pxl_wnd A PixelWand resource
blue The blue Quantum color value
