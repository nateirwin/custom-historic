MagickMapImage

MagickMapImage -- replaces the colors of the current active image with the closest color from a reference image

Description

bool MagickMapImage( MagickWand mgck_wnd, MagickWand map_wand, bool dither ) 

Replaces the colors of the current active image with the closest color
from a reference image.

A description of each parameter follows:

mgck_wnd A MagickWand resource
map The reference MagickWand -- the current active image in this MagickWand is used as the reference image.
dither Set this integer value to something other than zero to dither the mapped image
