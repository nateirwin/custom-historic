DrawSetFillRule

DrawSetFillRule -- sets the fill rule used while drawing polygons

Description

void DrawSetFillRule( DrawingWand drw_wnd, int fill_rule ) 

Sets the fill rule used while drawing polygons.

Returns void.

fill_rule must be a FillRule constant, one of the following:
   MW_EvenOddRule
   MW_NonZeroRule

A description of each parameter follows:

drw_wnd A DrawingWand resource
fill_rule A FillRule constant
