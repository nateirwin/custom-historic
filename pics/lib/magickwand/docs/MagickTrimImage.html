MagickTrimImage

MagickTrimImage -- removes edges that are the same color as the background color from the current active image

Description

bool MagickTrimImage( MagickWand mgck_wnd, float fuzz ) 

Removes edges that are the same color as the background color from the
current active image.

A description of each parameter follows:

mgck_wnd A MagickWand resource
fuzz By default target must match a particular pixel color exactly.
However, in many cases two colors may differ by a small amount.
The fuzz member of image defines how much tolerance is acceptable to consider two colors as the same.
For example, set fuzz to 10 and the color red at intensities of 100 and 102 respectively are now interpreted as the same color for the purposes of the floodfill
