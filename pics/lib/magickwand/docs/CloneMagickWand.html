CloneMagickWand

CloneMagickWand -- make a copy of a MagickWand

Description

MagickWand CloneMagickWand( MagickWand mgck_wnd ) 

Makes an exact copy of a MagickWand.

Returns the MagickWand copy, or FALSE if an error occurred.

A description of each parameter follows:

mgck_wnd The MagickWand resource to be cloned
