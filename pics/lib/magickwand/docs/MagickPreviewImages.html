MagickPreviewImages

MagickPreviewImages -- tiles 9 thumbnails of the specified image with an image processing operation applied at varying strengths

Description

MagickWand MagickPreviewImages( MagickWand mgck_wnd, PreviewType preview ) 

Tiles 9 thumbnails of the specified image with an image processing
operation applied at varying strengths.

This is helpful to quickly pin-point an appropriate parameter for an image
processing operation.

Returns a MagickWand resource containing the tiled image.

preview must be a PreviewType constant, one of the following:
   MW_RotatePreview        MW_ShearPreview
   MW_RollPreview          MW_HuePreview
   MW_SaturationPreview    MW_BrightnessPreview
   MW_GammaPreview         MW_SpiffPreview
   MW_DullPreview          MW_GrayscalePreview
   MW_QuantizePreview      MW_DespecklePreview
   MW_ReduceNoisePreview   MW_AddNoisePreview
   MW_SharpenPreview       MW_BlurPreview
   MW_ThresholdPreview     MW_EdgeDetectPreview
   MW_SpreadPreview        MW_SolarizePreview
   MW_ShadePreview         MW_RaisePreview
   MW_SegmentPreview       MW_SwirlPreview
   MW_ImplodePreview       MW_WavePreview
   MW_OilPaintPreview      MW_CharcoalDrawingPreview
   MW_JPEGPreview

A description of each parameter follows:

mgck_wnd A MagickWand resource
preview A PreviewType constant
