DrawPathStart

DrawPathStart -- declares the start of a path drawing list

Description

void DrawPathStart( DrawingWand drw_wnd ) 

Declares the start of a path drawing list which is terminated by a
matching DrawPathFinish() command.

All other DrawPath commands must be enclosed between a DrawPathStart() and
a DrawPathFinish() command.

This is because path drawing commands are subordinate commands and they do
not function by themselves.

A description of each parameter follows:

drw_wnd A DrawingWand resource
