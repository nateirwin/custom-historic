DrawSetStrokeDashArray

DrawSetStrokeDashArray -- specifies the pattern of dashes and gaps used to stroke paths

Description

bool DrawSetStrokeDashArray( DrawingWand drw_wnd [, array dash_array] ) 

Specifies the pattern of dashes and gaps used to stroke paths.

The dash_array parameter is an array of numbers (doubles) that specify the
lengths of alternating dashes and gaps in pixels.

If an odd number of values is provided, then the list of values is
repeated to yield an even number of values.

To remove an existing dash array, do not include the dash_array parameter,
or pass NULL or an empty array as dash_array.

A typical dash_array array might be array( 5, 3, 2 ).

A description of each parameter follows:

drw_wnd A DrawingWand resource
dash_array (OPTIONAL) an array of doubles representing the dash values.
If not specified, set as NULL, or as an empty array, the DrawingWand's dash values will be cleared
