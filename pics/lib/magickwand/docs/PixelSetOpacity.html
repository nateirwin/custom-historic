PixelSetOpacity

PixelSetOpacity -- sets the normalized opacity color value of the PixelWand

Description

void PixelSetOpacity( PixelWand pxl_wnd, float opacity ) 

Sets the normalized opacity color value of the PixelWand.

The opacity color value must be in the range 0 <= opacity <= 1.

A description of each parameter follows:

pxl_wnd A PixelWand resource
opacity The normalized opacity color value
