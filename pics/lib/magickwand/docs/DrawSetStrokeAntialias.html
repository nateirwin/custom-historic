DrawSetStrokeAntialias

DrawSetStrokeAntialias -- controls whether stroked outlines are antialiased

Description

void DrawSetStrokeAntialias( DrawingWand drw_wnd [, bool stroke_antialias] ) 

Controls whether stroked outlines are antialiased.

Stroked outlines are antialiased by default.

Also, if "stroke_antialias" argument is not specified, antialiasing is
turned on.

When antialiasing is disabled stroked pixels are thresholded to determine
if the stroke color or underlying canvas color should be used.

A description of each parameter follows:

drw_wnd A DrawingWand resource
stroke_antialias (OPTIONAL) set to FALSE to disable antialiasing.
If not set or TRUE, will turn on antialiasing
