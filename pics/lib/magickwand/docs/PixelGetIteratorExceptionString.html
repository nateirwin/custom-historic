PixelGetIteratorExceptionString

PixelGetIteratorExceptionString -- returns the description of a PixelIterator error

Description

string PixelGetIteratorExceptionString( PixelIterator pxl_iter ) 

Returns a string description of errors that occur when using other
PixelIterator methods in the API.

If there was no error present, an empty string ('') is returned.

A description of each parameter follows:

pxl_iter A PixelIterator resource
