DrawSetFontStretch

DrawSetFontStretch -- sets the font stretch used when annotating with text

Description

void DrawSetFontStretch( DrawingWand drw_wnd, int stretch_type ) 

Sets the font stretch used when annotating with text.

Returns void.

stretch_type must be a StretchType constant, one of the following:
   MW_NormalStretch              MW_UltraCondensedStretch
   MW_ExtraCondensedStretch      MW_CondensedStretch
   MW_SemiCondensedStretch       MW_SemiExpandedStretch
   MW_ExpandedStretch            MW_ExtraExpandedStretch
   MW_UltraExpandedStretch       MW_AnyStretch

A description of each parameter follows:

drw_wnd A DrawingWand resource
stretch_type A StretchType constant
