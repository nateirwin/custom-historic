DrawRender

DrawRender -- renders all preceding drawing commands

Description

bool DrawRender( DrawingWand drw_wnd ) 

Renders all preceding drawing commands.

Further drawing commands cannot be added (i.e., further calls to the
Draw...() functions).

A description of each parameter follows:

drw_wnd A DrawingWand resource
