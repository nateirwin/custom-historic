NewPixelWand

NewPixelWand -- returns a new PixelWand resource.

Description

PixelWand NewPixelWand( [string imagemagick_col_str] ) 

Returns a new PixelWand resource.

The optional imagemagick_col_str argument can be supplied to set the color
of the PixelWand.

imagemagick_col_str must be a valid ImageMagick color string (e.g.:
"blue", "#0000ff", "rgb(0,0,255)", "cmyk(100,100,100,10)", etc.), or a
fatal error will occur.

A description of each parameter follows:

imagemagick_col_str A valid ImageMagick color string, which will set the color of the returned PixelWand resource
