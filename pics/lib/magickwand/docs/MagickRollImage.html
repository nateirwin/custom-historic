MagickRollImage

MagickRollImage -- offsets the current active image

Description

bool MagickRollImage( MagickWand mgck_wnd, int x_offset, int y_offset ) 

Offsets the current active image as defined by x_offset and y_offset.

A description of each parameter follows:

mgck_wnd A MagickWand resource
x_offset The x offset.
y_offset The y offset
