MagickSetImageOption

MagickSetImageOption -- associates one or options with a particular image format

Description

bool MagickSetImageOption( MagickWand mgck_wnd, string format, string key, string value ) 

Associates one or options with a particular image format, e.g.:

   MagickSetImageOption( $mgck_wnd, "jpeg", "perserve", "yes").

A description of each parameter follows:

mgck_wnd A MagickWand resource
format The image format.
key The key.
value The value
