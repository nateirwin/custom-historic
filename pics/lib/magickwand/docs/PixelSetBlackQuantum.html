PixelSetBlackQuantum

PixelSetBlackQuantum -- sets the black Quantum color of the PixelWand

Description

void PixelSetBlackQuantum( PixelWand pxl_wnd, float black ) 

Sets the black Quantum color value of the PixelWand.

The black Quantum color value must be in the range 0 <= black <=
MW_MaxRGB.

A description of each parameter follows:

pxl_wnd A PixelWand resource
black The black Quantum color value
