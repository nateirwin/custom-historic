MagickSetImageProfile

MagickSetImageProfile -- adds a named profile to the current active image

Description

bool MagickSetImageProfile( MagickWand mgck_wnd, string name, string profile ) 

Adds a named profile to the current active image.

If a profile with the same name already exists, it is replaced.

This method differs from the MagickProfileImage() method in that it does
not apply any CMS color profiles.

A description of each parameter follows:

mgck_wnd A MagickWand resource
name Name of profile to add or remove: ICC, IPTC, or generic profile.
profile The profile
