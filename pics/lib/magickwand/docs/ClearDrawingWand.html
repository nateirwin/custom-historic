ClearDrawingWand

ClearDrawingWand -- clears a DrawingWand resource

Description

void ClearDrawingWand( DrawingWand drw_wnd ) 

Clears a DrawingWand resource of any accumulated commands, and resets the
settings it contains to their defaults.

A description of each parameter follows:

drw_wnd The DrawingWand resource to be cleared
