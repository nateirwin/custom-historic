DrawGetExceptionString

DrawGetExceptionString -- returns the description of a DrawingWand error

Description

string DrawGetExceptionString( DrawingWand drw_wnd ) 

Returns a string description of errors / exceptions that occur when using
other DrawingWand methods in the API.

If there is no error present, an empty string ('') is returned.

A description of each parameter follows:

drw_wnd A DrawingWand resource
