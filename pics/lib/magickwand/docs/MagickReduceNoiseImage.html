MagickReduceNoiseImage

MagickReduceNoiseImage -- smooths the contours of the current active image

Description

bool MagickReduceNoiseImage( MagickWand mgck_wnd, float radius ) 

Smooths the contours of the current active image while still preserving
edge information.

The algorithm works by replacing each pixel with its neighbor closest in
value.

A neighbor is defined by radius.

Use a radius of 0 and ReduceNoise() selects a suitable radius for you.

A description of each parameter follows:

mgck_wnd A MagickWand resource
radius The radius of the pixel neighborhood
