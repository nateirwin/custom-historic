MagickSetImageIndex

MagickSetImageIndex -- sets the MagickWand's image index to the specified position

Description

bool MagickSetImageIndex( MagickWand mgck_wnd, int index ) 

Sets the MagickWand's image index to the specified position; the current
active image within the MagickWand becomes the image at the specified
index.

Returns TRUE if successful, and FALSE if there are no images in the
MagickWand, if the requested image index does not exist, or if some other
error occurred.

A description of each parameter follows:

mgck_wnd A MagickWand resource
index The requested index
