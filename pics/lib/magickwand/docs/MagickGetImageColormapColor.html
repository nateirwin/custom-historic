MagickGetImageColormapColor

MagickGetImageColormapColor -- returns the color of the specified colormap index of the current active image

Description

PixelWand MagickGetImageColormapColor( MagickWand mgck_wnd, float index ) 

Returns the color of the specified colormap index of the current active
image, as a PixelWand.

A description of each parameter follows:

mgck_wnd A MagickWand resource
index The offset into the image colormap
