MagickEchoImageBlob

MagickEchoImageBlob -- outputs the current active image to stdout as a BLOB

Description

bool MagickEchoImageBlob( MagickWand mgck_wnd ) 

Outputs the BLOB (Binary Large OBject) of the current active image in the
MagickWand to stdout, which on the web whould be the user's browser.

The format of the image determines the format of the output blob (GIF,
JPEG, PNG, etc.).

To output the image BLOB in a different image format, use
MagickSetImageFormat().

Returns TRUE on success, FALSE if an error occurred.

A description of each parameter follows:

mgck_wnd A MagickWand resource
