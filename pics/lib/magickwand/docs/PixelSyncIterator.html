PixelSyncIterator

PixelSyncIterator -- syncronizes the PixelIterator

Description

bool PixelSyncIterator( PixelIterator pxl_iter ) 

Syncronizes the PixelIterator with the image (in the MagickWand) from
which it was derived.

Essentially, changes made to the PixelWands which make up the rows through
which the PixelIterator iterates, will be applied to the pixels in the
image (in the MagickWand) on which the PixelIterator was based.

Will return TRUE if successful, FALSE if otherwise.

A description of each parameter follows:

pxl_iter A PixelIterator resource
