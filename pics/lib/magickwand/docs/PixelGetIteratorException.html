PixelGetIteratorException

PixelGetIteratorException -- returns the description and type of a PixelIterator error

Description

array PixelGetIteratorException( PixelIterator pxl_iter ) 

Returns an array containing the severity and description of errors that
occur when using other PixelIterator methods in the API.

The returned array is defined as follows:
Array
{
   [0] => (ExceptionType) type: The (ErrorType constant) type of the
          error.
          If no error has occurred, type == MW_UndefinedException.

   [1] => (string) description: The error description (if no error,
          description == '')
}

A description of each parameter follows:

pxl_iter A PixelIterator resource
