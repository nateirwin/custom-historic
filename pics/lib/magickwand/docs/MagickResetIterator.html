MagickResetIterator -- resets the MagickWand's image iterator

Description

void MagickResetIterator( MagickWand mgck_wnd ) 

Resets the MagickWand's image iterator.

Use it in conjunction with MagickNextImage() to iterate over all the
images in the MagickWand's image list.

A description of each parameter follows:

mgck_wnd A MagickWand resource
