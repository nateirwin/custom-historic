MagickAnnotateImage

MagickAnnotateImage -- annotates the current active image with text

Description

bool MagickAnnotateImage( MagickWand mgck_wnd, DrawingWand drw_wnd, float x, float y, float angle, string text ) 

Annotates the current active image with text.

The text's attributes, (font, color, etc.), will be set by the DrawingWand
argument.

A description of each parameter follows:

mgck_wnd A MagickWand resource
drawing_wand A DrawingWand resource.
x X ordinate to left of text
y Y ordinate to text baseline
angle Rotate text relative to this angle.
text Text to draw
