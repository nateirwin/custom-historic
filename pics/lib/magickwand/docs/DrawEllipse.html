DrawEllipse -- draws an ellipse on the image

Description

void DrawEllipse( DrawingWand drw_wnd, float ox, float oy, float rx, float ry, float start, float end ) 

Draws an ellipse on the image.

A description of each parameter follows:

drw_wnd A DrawingWand resource
ox Origin x ordinate
oy Origin y ordinate
rx Radius in x
ry Radius in y
start Starting rotation in degrees
end Ending rotation in degrees
