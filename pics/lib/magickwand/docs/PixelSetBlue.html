PixelSetBlue

PixelSetBlue -- sets the normalized blue color value of the PixelWand

Description

void PixelSetBlue( PixelWand pxl_wnd, float blue ) 

Sets the normalized blue color value of the PixelWand.

The blue color value must be in the range 0 <= blue <= 1.

A description of each parameter follows:

pxl_wnd A PixelWand resource
blue The normalized blue color value
