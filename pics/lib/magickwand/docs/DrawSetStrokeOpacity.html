DrawSetStrokeOpacity

DrawSetStrokeOpacity -- specifies the opacity of stroked object outlines

Description

void DrawSetStrokeOpacity( DrawingWand drw_wnd, float stroke_opacity ) 

Specifies the opacity of stroked object outlines.

A description of each parameter follows:

drw_wnd A DrawingWand resource
stroke_opacity Stroke opacity.
The value 1.0 is opaque
