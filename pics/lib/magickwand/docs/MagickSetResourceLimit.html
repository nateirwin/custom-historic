MagickSetResourceLimit

MagickSetResourceLimit -- sets the limit for a particular resource in megabytes

Description

bool MagickSetResourceLimit( int resource_type, float limit ) 

Sets the limit for a particular resource in megabytes.

Returns TRUE on success, FALSE on failure.

resource_type must be a ResourceType constant, one of the following:
   MW_AreaResource
   MW_DiskResource
   MW_FileResource
   MW_MapResource
   MW_MemoryResource

A description of each parameter follows:

resource_type A ResourceType constant
limit The maximum limit for the resource
