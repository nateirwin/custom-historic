DrawSetTextAlignment

DrawSetTextAlignment -- sets the alignment that will be applied when annotating with text

Description

void DrawSetTextAlignment( DrawingWand drw_wnd, int align_type ) 

Sets the alignment that will be applied when annotating with text.

Returns void.

align_type must be an AlignType constant, one of the following:
   MW_LeftAlign
   MW_CenterAlign
   MW_RightAlign

A description of each parameter follows:

drw_wnd A DrawingWand resource
align_type An AlignType constant
